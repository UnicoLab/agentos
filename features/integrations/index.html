    Integrations¶

    Connect AgentOS to your existing development tools to aggregate project intelligence into one unified view. Direct API connections from your machine — no middleware, no proxy, no data routing.

    Setup Guides → Download AgentOS

    GitHub¶

    Multi-repository support with deep analytics. Aggregate data from all your repos into a single project view:

    Pull Requests¶

    Status tracking, reviewer analysis, stale PR detection, and AI-powered reviewer suggestions.

    Commits¶

    Activity charts, per-author breakdown, velocity trends, and contribution heatmaps.

    Issues¶

    Complete issue tracking with labels, priorities, milestones, and automated categorization.

    Contributors¶

    Activity metrics, velocity per developer, leaderboard ranking, and engagement patterns.

    Setup Guide →

    Jira¶

    Multi-board aggregation for complete sprint and project tracking:

    Sprint Status¶

    WIP, Done, Todo ring charts with real-time sprint progress tracking across all boards.

    Epic Tracker¶

    Progress bars per epic with grouped issue views and roll-up metrics.

    Mind Map¶

    Visual issue hierarchy and dependency graph for your entire backlog.

    Sprint Planner¶

    AI-generated sprint plans and milestones based on team velocity and capacity.

    Setup Guide →

    Slack¶

    Push reports and intelligence directly to your team channels:

    • Standup Reports — One-click send to any Slack channel
    • Risk Alerts — Automated notifications about blockers and critical risks
    • Scheduled Delivery — Automated daily/weekly summaries on your cadence

    Setup Guide →

    AIFlow Platform¶

    Connect to the centralized AIFlow platform for organization-wide intelligence:

    • Push Reports — Share structured insights with your team on the central hub
    • Multi-Agent Orchestration — Connect to AIFlow's agentic workflows and automation
    • Centralized Dashboard — View all projects across the organization

    Setup Guide →

    AI Providers¶

    Choose your AI model — switch anytime without losing data or configuration:

    Provider Type Models Privacy
    Ollama Local Llama 3.2, Mistral, CodeLlama ✅ 100% local — prompts never leave your machine
    OpenAI Cloud GPT-4o, GPT-4o-mini API calls to OpenAI servers
    Anthropic Cloud Claude 3.5 Sonnet, Opus API calls to Anthropic servers
    Google Cloud Gemini 2.0 Flash, Pro API calls to Google servers

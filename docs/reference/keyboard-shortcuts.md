# :material-keyboard: Keyboard Shortcuts

Master these shortcuts to navigate AgentOS like a pro — every action is a keystroke away.

</div>

---

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| ++cmd+k++ | Open Command Palette |
| ++cmd+m++ | Open JP Memory panel |
| ++cmd+n++ | New chat session |
| ++cmd+t++ | Tool Explorer |
| ++cmd+comma++ | Provider Settings |
| ++cmd+r++ | Refresh dashboard data |
| ++slash++ | Focus chat input |
| ++question++ | Show keyboard shortcuts reference |
| ++escape++ | Close any open panel |

---

## Navigation

| Shortcut | Action |
|----------|--------|
| ++cmd+b++ | Toggle sidebar |
| ++cmd+1++ | Switch to Dashboard |
| ++cmd+2++ | Switch to Chat |

---

## Chat

| Shortcut | Action |
|----------|--------|
| ++enter++ | Send message |
| ++shift+enter++ | New line in chat input |
| ++up++ | Edit last message |
| ++cmd+n++ | Start new session |

---

!!! tip "Power User Tip"
    The **Command Palette** (++cmd+k++) is your fastest way to navigate. Type any action, page, or command — it's the Spotlight/Alfred equivalent for AgentOS.

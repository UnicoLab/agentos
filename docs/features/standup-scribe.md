## The Problem

Every morning starts the same way: spend 15–20 minutes pulling together what happened yesterday, what's planned today, and what's blocked. Check GitHub for merged PRs. Check Jira for sprint items. Check Slack for overnight updates. Then condense it all into a coherent message. **Every. Single. Day.**

---

## The Solution

<div class="screenshot" markdown>
![Standup Scribe — AI-generated daily standup with structured sections](../assets/screens/flavors/pm/standup.png)
</div>

Click one button. JP pulls real activity data from GitHub and Jira, structures it into **Yesterday / Done**, **Today / In Progress**, and **Blockers / Risks** — tags each item by type (FEATURE, FIX, OPS, TECH, DEPENDENCY) — and saves it as a project note. Done.

!!! success "Impact"
    **Standup prep goes from 20 minutes to 5 seconds.** And the report is more accurate than anything you'd write manually.

# Integrations

AgentOS connects to your existing development tools to aggregate project intelligence.

---

## GitHub

Multi-repository support with deep analytics:

- **Pull Requests** — Status, reviewers, AI-powered reviewer suggestions
- **Commits** — Activity charts, per-author breakdown, trend analysis  
- **Issues** — Tracking with labels and priority
- **Contributors** — Activity metrics, velocity, leaderboard
- **Webhooks** — Real-time push events

[Setup Guide →](../guides/github-setup.md)

---

## Jira

Multi-board aggregation for sprint and project tracking:

- **Sprint Status** — WIP, Done, Todo ring charts
- **Issues** — By type, priority, label, and epic
- **Epic Tracker** — Progress bars per epic
- **Mind Map** — Visual issue hierarchy
- **Sprint Planner** — AI-generated sprint plans

[Setup Guide →](../guides/jira-setup.md)

---

## Slack

Send reports and standup summaries to your team channels:

- **Standup Reports** — One-click send to Slack
- **Notifications** — Alert channels about risks and blockers

[Setup Guide →](../guides/slack-setup.md)

---

## AIFlow Platform

Sync project intelligence with the centralized AIFlow platform:

- **Push Reports** — Share insights with your team
- **Multi-agent Orchestration** — Connect to AIFlow's agentic workflows
- **Centralized Dashboard** — View all projects across the organization

[Setup Guide →](../guides/aiflow-sync.md)

## Dynamic Card Library

The dashboard includes a comprehensive set of widgets that auto-populate with your live project data:

| Card | Description |
|------|-------------|
| :material-chart-box: **KPIs** | Open PRs, issues, commits, stars — dynamic labels based on selected date range |
| :material-source-pull: **Pull Requests** | Live PR list with AI-powered reviewer suggestions and age tracking |
| :material-puzzle: **Sprint Status** | Jira metrics visualized as progress rings (WIP vs Done vs Todo) |
| :material-sitemap: **Mind Map** | Visual dependency graph of Jira issues, epics, and relationships |
| :material-trophy: **Leaderboard** | Gamified team contribution metrics with streaks and badges (🔥🥇👑) |
| :material-timeline: **Activity Feed** | Reverse-chronological timeline of all project events |
| :material-chart-line: **Commit Activity** | Per-author commit charts with trend analysis |
| :material-chart-areaspline: **PR Timeline** | Visual PR lifecycle from open to merge |
| :material-fire: **Contribution Heatmap** | 26-week contribution heatmap per author |
| :material-pin: **Pinned Insights** | Carousel of past AI analyses and saved recommendations |
| :material-alert: **Actionable Alerts** | AI-detected risks with auto-fix suggestions |
| :material-factory: **Production Pipelines** | One-click report generators (Weekly, Risk, Standup, Sprint) |

## Keyboard Mastery

Press ++cmd+k++ anytime to open the **Command Palette**.

| Action | Shortcut |
|--------|----------|
| **Refresh Data** | ++cmd+r++ |
| **Search Engine** | ++cmd+k++ |
| **Toggle Sidebar** | ++cmd+b++ |
| **Agent Settings** | ++cmd+comma++ |

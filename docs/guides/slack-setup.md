## 🏗️ Connectivity

<div class="step-grid" markdown>

<div class="step-card" markdown>
<div class="step-num">1</div>
### Create App
Visit [api.slack.com](https://api.slack.com/apps) and create a new app from scratch.
</div>

<div class="step-card" markdown>
<div class="step-num">2</div>
### Add Webhook
Enable **Incoming Webhooks** and authorize a channel (e.g., `#standups`).
</div>

<div class="step-card" markdown>
<div class="step-num">3</div>
### Configure
Paste the Webhook URL into the **Settings > Integrations > Slack** panel.
</div>

</div>
